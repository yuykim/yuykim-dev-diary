# 왜 이 작업을 했는가

관절 중심 텔레옵만으로는 정밀한 Cartesian 이동이 어렵다. 방향키로 X/Y/Z를 직접 조작할 수 있으면 데이터 수집 품질이 높아진다.

# 작업 과정

**discard 기능:** `data_collector_node.py`에 `/data_collector/discard` 서비스 추가. 녹화 종료 시 `[Y] Success / [N] Fail / [X] Discard` 선택지 추가.

**Cartesian 텔레옵:**
- `/servo_node/start_servo` 서비스 호출 후 `/servo_node/delta_twist_cmds`로 `TwistStamped` publish
- 키 매핑: 방향키(X/Y), `;/.`(Z), `B/T`(base/TCP frame), `9/0`(속도 조절)
- `data_collection.launch.py`에 `enable_cartesian_servo` 인자 추가 (기본 false)
