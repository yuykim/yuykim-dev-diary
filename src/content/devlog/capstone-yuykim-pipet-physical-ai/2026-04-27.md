# 오늘 한 일

- MuJoCo에서 Indy7 + Mark7 통합 모델 구성
- 키보드 Cartesian 텔레옵 스크립트 추가
- Mark7 실제 xacro 기반 결합 및 프리셋 조작 반영

# 왜 이 작업을 했는가

실제 로봇 없이도 시뮬레이션에서 동작을 검증할 수 있어야 개발 속도가 빨라진다. 특히 MuJoCo 환경에서 Indy7+Mark7 통합 뷰어가 없었음.

# 작업 과정

**모델 결합:** `prepare_models.py` 확장 - Indy7 URDF + Mark7 xacro 결합 → `generated/indy7_mujoco.urdf`. Mark7 링크/조인트에 `mark7_` prefix 부여해 충돌 회피. `link6`에 `mark7_mount_joint`(fixed)로 결합.

**텔레옵 스크립트:** `keyboard_cartesian_teleop.py` 추가.
- W/S(+/-X), A/D(+/-Y), Q/E(+/-Z), B/T(world/tool frame), `[`/`]`(속도)
- kinematic 적분 기반 제어 (정지 시 처짐 방지)

**Mark7 프리셋:** `grip_presets.yaml` 기준으로 G/O/P/R 동작 매핑. step → rad 변환 후 MuJoCo 관절 목표값 적용. mimic 관절 수동 동기화.

# 중간 결과

- MuJoCo 장면에서 Indy7+Mark7 함께 로드 성공
- 키보드로 Cartesian 이동 + 그리퍼 프리셋 조작 가능
- Mark7 소스 경로 자동 탐색 (ros2_ws → external 순서)

# 막힌 점

- MuJoCo에서 mimic 관절이 자동 적용 안 됨 → 수동 동기화 코드 추가
- 장착 자세(`mark7_mount_joint` xyz/rpy)는 미세 튜닝 필요

# 느낀 점

시뮬레이터가 생기니까 하드웨어 없이도 알고리즘을 테스트할 수 있다는 안도감이 생긴다. 실제 물리는 다르지만 동작 검증에 유용하다.
