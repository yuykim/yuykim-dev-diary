# 프로젝트 소개

이 글은 기존 개발 repo를 yuykim14 Dev Diary에 처음 연결하면서 생성한 프로젝트 소개 글이다.

`gfootball-RLfootball-Imitation_Learning`은 Google Research Football 환경에서 reinforcement learning과 imitation learning을 실험한 프로젝트다. 앞으로 이 프로젝트의 `dev_diary/` 기록이 추가되면 같은 프로젝트 타임라인에 이어서 쌓인다.

# 기록 방식

- `HISTORY.md`는 작업 이유, 실행 기록, 디버깅 기록, 되돌림 판단 기준을 남기는 용도로 사용한다.
- `dev_diary/YYYY-MM-DD.md`는 블로그에 올라갈 개발 일기 원본으로 사용한다.
- 이미지와 실행 결과는 `dev_diary/assets/`에 저장한다.
