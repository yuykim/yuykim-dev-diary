# 시작점

Neuromeka Indy7 로봇팔을 Python과 Pygame으로 키보드 제어하기 위한 프로젝트를 Dev Diary에 연결했다.

이 repo는 로봇 상태 확인, 에러 복구, 홈 포지션 이동, 6자유도 키보드 teleoperation 흐름을 정리한다. 블로그에는 하드웨어 세팅 자체보다 제어 코드 구조, 안전하게 조작하기 위한 흐름, 실험 중 생긴 문제를 중심으로 기록한다.

## 기록할 내용

- Indy7 연결 확인과 실행 전 체크
- Pygame 기반 키 입력 처리 구조
- 위치/회전 제어 키맵 변경
- 에러 복구와 종료 루틴
- 실험 중 발견한 조작 안정성 문제

## 지금까지 한 작업 요약

- `indy7_start.py`로 로봇 연결, 서보 활성화, 상태 확인 흐름을 만들었다.
- `error.py`로 teleop 중단, recover, servo on, 에러 상태 확인을 묶은 복구 루틴을 만들었다.
- `indy7_shutdown.py`로 home 복귀, motion stop, servo off를 수행하는 종료 루틴을 만들었다.
- `indy7_keyboard_control_v1.py`에서 Pygame 입력 루프와 `IndyDCP3` 상대 task teleop을 연결했다.
- 키보드로 X/Y/Z 이동과 RX/RY/RZ 회전을 제어하고, `[`/`]`로 step size를 조절하게 했다.
- `check_teleop.py`와 `check_controller.py`로 teleop 상태와 외부 컨트롤러 입력을 진단했다.
- `readme.md`에 네트워크 설정, 실행 방법, 키맵, 안전 주의사항, 참고 이미지를 정리했다.
