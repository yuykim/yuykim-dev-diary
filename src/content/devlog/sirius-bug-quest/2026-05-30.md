# 시작점

Unity 기반 Bug Quest 프로젝트를 Dev Diary에 연결했다.

이 repo는 게임플레이 구현, 버그 수정, 사운드/애셋 반영, 빌드 전후의 반복 작업을 기록하기 좋다. 앞으로는 기능을 추가하거나 문제를 고칠 때 단순 변경 목록보다 어떤 판단으로 수정했는지를 diary에 남긴다.

## 기록할 내용

- 플레이어와 몬스터 동작 구현
- 보스전, 사운드, 배경 애셋 반영
- Unity 프로젝트 설정이나 빌드 문제
- 팀 작업 중 생긴 병합/디버깅 이슈
- 출시 또는 제출용 빌드 준비 과정
